[ PHILOSOPHIE ]

BiCrypt n'est pas une application de messagerie classique. C'est un outil de communication souverain qui refuse toute dépendance aux infrastructures centralisées. Pas de serveur, pas de base de données cloud, pas de point de défaillance unique.

[ ARCHITECTURE ]

En combinant le réseau Tor pour l'anonymat global et un mesh Bluetooth Low Energy pour la proximité, BiCrypt garantit que vos messages atteignent leur destination, que vous soyez connecté au web ou en isolation totale.

Cryptographie X25519 / XSalsa20
Réseau Tor V3 / BLE Mesh
